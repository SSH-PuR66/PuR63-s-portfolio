### CivicPulse – Open-Government Data Dashboard

Developed an open-government data dashboard that makes U.S. Census Bureau statistics accessible and visually digestible.

- Consumed ACS 5-Year API data asynchronously using httpx
- Normalized raw FIPS-coded responses into clean state-level records
- Displayed total population and median household income data
- Built a TTL-based in-memory cache layer to reduce redundant API calls
- Developed backend with FastAPI using router/service separation
- Packaged the application in a production-ready Docker container
- Used GitLab CI/CD with SAST and secret-detection jobs
- Built interactive frontend data visualizations with Chart.js
- Added accessibility-focused features including ARIA live regions and skip-navigation links

Repository: <a href="https://github.com/SSH-PuR66">View on GitHub</a>
